# SoulForge

SoulForge is an AI-powered terminal-based IDE (Integrated Development Environment) built using Bun, a fast all-in-one JavaScript runtime.

## Features

- **AI-Powered Assistance**: SoulForge integrates with large language models (LLMs) to provide intelligent code suggestions, refactoring, and debugging assistance.
- **Terminal-Based Interface**: SoulForge runs entirely within the terminal, providing a seamless and efficient development experience.
- **Bun Runtime**: The project is built using Bun, a fast and modern JavaScript runtime that provides a streamlined development workflow.
- **Extensible with Skills**: SoulForge supports the ability to load "skills" - custom modules that extend the functionality of the IDE.

## Getting Started

To get started with SoulForge, follow these steps:

1. Install the dependencies:
   ```bash
   bun install
   ```
2. Run the application:
   ```bash
   bun run src/index.tsx
   ```

This will start the SoulForge IDE, ready for you to begin your development journey.
